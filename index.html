How to Connect Google Sheets
1

Create a Google Sheet

Open Google Sheets → create a new sheet. Name the first sheet Attendance. Add these headers in row 1:

ID | Name | Department | Role | Date | Time | Status | Day
2

Open Apps Script

In your Google Sheet: Extensions → Apps Script. Delete any existing code and paste the script below:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getSheetByName("Attendance");
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.id,
    data.name,
    data.department,
    data.role,
    data.date,
    data.time,
    data.status,
    data.day
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({result:"success"}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getSheetByName("Attendance");
  var data = sheet.getDataRange().getValues();
  return ContentService
    .createTextOutput(JSON.stringify({data: data}))
    .setMimeType(ContentService.MimeType.JSON);
}
3

Deploy as Web App

Click Deploy → New Deployment. Select type: Web App. Set "Execute as" = Me, "Who has access" = Anyone. Click Deploy, authorize, and copy the Web App URL.

4

Paste URL in Admin Tab

Go to the Admin tab → paste the Web App URL and click Save. The system will now log attendance directly to your Google Sheet.
